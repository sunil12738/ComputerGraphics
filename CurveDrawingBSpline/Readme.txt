To run the assignment:
----------------------
Use the bash script "compile.sh" to compile and run the assignment.

Things which can be done:
-------------------------
1. User can draw one or more than one curves. To draw a new curve press 'n' key from keyboard
2. User can change the color of the curves while drawing.
	Press the following keys to get the desired curves
	'r':red
	'g':green
	'b':blue
3. We have displayed the control polygon on the current working curve.
